# 全局安装/运行/打包命令

本项目之前vuepress为全局安装并且为0.X版本，所有项目包内未引用。命令如下

# 安装
npm install -g vuepress@0.14.11
# 运行
npm run docs:dev
# 打包
npm run docs:build
