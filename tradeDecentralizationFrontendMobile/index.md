## 项目发版分支

此项目发版为release_ALL(全部功能的分支)和release_hasNotTradeRoute（带有部分功能的分支(去除界面上所有的交易及其配套按钮和路由跳转)）两个分支

## 项目发版版本命令

* 本地测试版本 
  
``` bash
npm run localsever
```

* 线上测试版本  

``` bash
npm run alpha
```

* 线上正式版本  


``` bash
npm run beta
```
